# reactNativeTest

# How to setup this project on system ?

# for IOS 
 Do run 
 - npm install 

 then to run from terminal run
 - react-native run-ios

 or you need to configure xcode

# for android 
Do run
- npm install
- react-native run-android

if server getting disconnected
then run 
- adb reverse tcp:8081 tcp:8081
- lsof -i:8081
- npm start
